seg2class

Easily transform segmentation datasets into classification ones.

Start immediately crunching data, customize and extend later to create the perfect dataset.

Faster dataset conversion

From segmentation datasets

Datasets used to train neural nets for segmentation tasks are comprised of two things: source images and masks.

seg2class is a small Python 3.x library for generating a set of images that can be used to train classification models. This is achieved by extracting patches from the input images and classifying them based on the segmentation masks.

Works out of the box

seg2class works without needing any particular configuration, making few assumptions on the dataset's structure and mask type: if your case isn't covered, you can always customize it.

Specify a master folder, containing two subfolders: images and masks. Inside these, there should be as many folders as your classes, each one containing the images or the masks pertaining that class. seg2class supports all primary image formats like jpg, png, tiff and bmp.

Free and open source

seg2class is distributed under the MIT license. Feel free to use, modify and extend it even for commercial purposes - credit not needed but appreciated.

By the way, contributions are always welcome!

How to use seg2class

Clone from Github

Get the code

Download or clone from GitHub the seg2class repository, using either https or SSH.

git clone https://github.com/marcopinato/seg2class.git

Install dependencies and organize data

Setup

Organize the segmentation dataset as described in the README and install dependencies.

pip3 install -r requirements.txt

Open a terminal and launch

Create a classification dataset

Start the conversion by passing the segmentation dataset's path.

python3 seg2class.py /home/marco/dataset

Create your classification dataset

and then go back worrying about those vanishing gradients.
